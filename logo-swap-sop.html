Standard Operating Procedure — Seasonal Logo Swap  |  Last updated March 2026

When you replace the logo image in GitHub, every employee's email signature updates automatically — no HTML changes needed.

Requirements: The new image must be saved as exactly adas-logo.png and must have a transparent background (PNG format).

Steps

1 Open your GitHub images folder

Click the link below to go directly to the images folder:

https://github.com/markfowler01/markfowler01.github.io/tree/main/absolute-adas/images

2 Click on adas-logo.png

You'll see the current logo file listed. Click on adas-logo.png to open it.

3 Upload the new logo

  1. In the top right area of the file view, click the three dots menu (⋯)
  2. Select "Replace file" (or click the upload icon)
  3. Choose your new adas-logo.png file from your computer

4 Save the change

  1. Scroll down to the Commit changes section
  2. Leave the message as-is or type something like "Update seasonal logo"
  3. Click Commit changes
  4. Click Commit changes again to confirm

5 Done — wait 2-3 minutes

GitHub Pages takes a couple of minutes to update. After that, anyone who opens an email from an Absolute ADAS employee will see the new logo automatically.

To verify it worked, open this URL in your browser:

https://absoluteadas.com/images/adas-logo.png
💡 Tip: Keep a folder of seasonal logos ready to go — Christmas, 4th of July, Thanksgiving, etc. Just make sure each one is saved as adas-logo.png with a transparent background before uploading.
