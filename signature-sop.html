Standard Operating Procedure — Employee Email Signatures  |  Last updated March 2026

Overview

All Absolute ADAS employees use a standardized HTML email signature in Zoho Mail. The signature includes the company logo, employee name, title, email, phone number, website, and tagline.

The logo is hosted permanently at:

https://absoluteadas.com/images/adas-logo.png

Seasonal updates: To swap the logo (e.g. for holidays), replace adas-logo.png in the absolute-adas GitHub repository under /images/. All employee signatures update automatically — no HTML changes needed.

Step 2 — Add the Signature in Zoho Mail

  1. Log into Zoho Mail at mail.zoho.com
  2. Click the gear icon (Settings) in the top right corner
  3. Click Signatures in the left menu
  4. Click Add New Signature
  5. Give it a name — e.g. Absolute ADAS
  6. In the signature editor, click the <> button to switch to HTML mode
  7. Delete any existing content in the box
  8. Paste the completed HTML code from Step 1
  9. Click Save

Step 3 — Set as Default Signature

  1. In Zoho Mail Settings, go to Mail Accounts
  2. Find your email account and click Edit
  3. Under Default Signature, select Absolute ADAS
  4. Click Save

Step 4 — Verify

Compose a new email to yourself and confirm:

  • Logo appears correctly
  • Name and title are correct
  • Email address and phone number are correct
  • Website link opens absoluteadas.com
